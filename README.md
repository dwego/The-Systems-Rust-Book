# The Systems Rust Book

### *Learn Rust through systems programming.*

An open-source book about Rust, low-level programming, embedded systems, and operating system development.

> **Built by learners, for learners.**

---

</div>

## About

**The Systems Rust Book** is an open-source project focused on teaching **Rust from a systems programming perspective**.

Instead of approaching Rust through web development or command-line applications, this book explores the language from the ground up—covering memory, ownership, `core`, `alloc`, `std`, `unsafe`, `no_std`, embedded development, and operating systems.

This is also a personal learning journey. Every chapter is written while studying, experimenting, and building real projects, with the goal of creating a resource that helps others follow the same path.

---

## What you'll learn

* Rust fundamentals
* Ownership, borrowing, and lifetimes
* Memory layout
* Stack vs Heap
* `core`, `alloc`, and `std`
* Unsafe Rust
* `no_std`
* Embedded Rust
* Memory-mapped I/O
* Interrupts
* Device drivers
* Boot process
* Operating system development

## Philosophy

This project follows a few simple principles.

* Learn by building.
* Understand what happens beneath the abstractions.
* Prefer safe Rust whenever possible.
* Document every important engineering decision.
* Build a strong foundation before moving to advanced topics.
